# Downloads Folder

Place your downloadable files here. They will be available at:
- `/downloads/filename.pdf` 
- etc.

## Expected Files

The Toolkit section expects these files:

1. `youth-count-flyer.pdf` - Official promotional flyer
2. `social-media-kit.zip` - ZIP containing social media graphics
3. `host-guide.pdf` - Magnet Site Host Guide

## How to Add Files

1. Export/save your files with the names above
2. Upload them to this folder in your GitHub repo
3. Push to GitHub - Vercel will automatically redeploy

## File Size Notes

- Files under 50MB work fine in GitHub
- For larger files, consider using Google Drive or Dropbox with public links
